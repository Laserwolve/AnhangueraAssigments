Este programa em Python implementa um sistema de gestão de notas de alunos desenvolvido para atender requisitos de programação estruturada. O objetivo é automatizar o cadastro de notas, cálculo de médias e determinação da situação acadêmica dos estudantes. O sistema possui cinco funções principais: adicionar_notas permite inserir múltiplas notas com validação entre 0 e 10, finalizando com -1 e tratando entradas inválidas automaticamente. A função calcular_media retorna a média aritmética das notas com proteção contra divisão por zero. A determinar_situacao aplica o critério de aprovação com média maior ou igual a 7.0 para aprovado e menor que 7.0 para reprovado usando estruturas condicionais. A exibir_relatorio apresenta todas as informações de forma organizada: notas inseridas, quantidade total, média com duas casas decimais, situação final e mensagem motivacional personalizada. A função main coordena o fluxo completo permitindo processar múltiplos alunos através de loop principal com tratamento de exceções para maior robustez. O código demonstra aplicação prática de conceitos fundamentais incluindo estruturas condicionais, repetição, funções, listas, validação de entrada e tratamento de erros, proporcionando interface amigável com mensagens claras e documentação em português seguindo boas práticas de programação estruturada.
